# Установка плагина

Последняя версия плагина: 2018.3.1. (Опубликовано 10 января 2019 г.)
**Примечание. Плагин Gitee IDEA был переименован в `gitee` из `gitosc`.**
**Меню Gitee плагина новой версии было объединено с меню Git**

## Установка через «Управление плагинами»

1. Запустите IDEA
2. На стартовом экране выберите меню «Настроить» -> «Плагины».
3. Найдите ключевое слово «Gitee» на рынке всплывающих плагинов и найдите плагин «Gitee» в результатах поиска. Нажмите «Установить», чтобы установить плагин.
4. Перезапустите IDEA, чтобы плагин заработал.

![](https://images.gitee.ru/uploads/images/2019/0111/164940_57b7170b_551147.png)

## Установка через «Настройки->Плагины»

1. Запустите IDEA и перейдите в «Настройки» (*В macOS откройте «Настройки» из меню, в Windows откройте «Настройки» из меню «Файл»*), выберите «Плагины».
2. Найдите ключевое слово «Gitee» на появившемся рынке плагинов. Найдите плагин «Gitee» в результатах поиска и нажмите «Установить», чтобы установить его.
3. Перезапустите IDEA, чтобы плагин заработал.

![](https://images.gitee.ru/uploads/images/2019/0111/165953_373293a9_551147.png)

## Установка с помощью загрузки пакета плагинов

Адрес плагина: [https://plugins.jetbrains.com/plugin/11491-gitee](https://plugins.jetbrains.com/plugin/11491-gitee). Загрузите напрямую и установите вручную.

Вы также можете скачать и установить его, открыв репозиторий плагина с открытым исходным кодом: [https://gitee.ru/oschina/intellij-gitosc](https://gitee.ru/oschina/intellij-gitosc).

Войдите и извлеките код репозитория

# Задачи, связанные с Gitee

Используя плагины, вы можете связать задачи IDEA с задачами Gitee.

1. Выберите «Инструменты» -> «Задачи и контексты» -> «Настроить серверы...».
2. Нажмите «+» и выберите «GitOSC», введите имя пользователя и имя репозитория. Эти два имени должны соответствовать именам в URL-адресе репозитория.
3. Нажмите «Создать токен API», введите свой адрес электронной почты и пароль, нажмите «Войти», и токен будет сгенерирован.
4. Наконец, нажмите «Подтвердить».

![](https://images.gitee.ru/uploads/images/2019/0111/172713_e432d911_551147.gif)

Откройте задачи следующим образом

1. Выберите «Инструменты» -> «Задачи и контексты» -> «Открыть задачу...».
2. Задачи репозитория отобразятся в списке, нажмите дважды.
3. Настройте ветки репозитория и т. д. и подтвердите в последний раз

![](https://images.gitee.ru/uploads/images/2019/0111/172714_e1c1793b_551147.gif)

**Управление задачами**

1. 'Открыть задачу...' установка связь между задачами.
2. `Закрыть активную задачу` обновление статуса задачи.
3. `Показать описание'***'` отображение информации об описании задачи.
4. `Открыть '***' в браузере` открывает страницу задачи в браузере.
![](https://images.gitee.ru/uploads/images/2019/0111/172714_9d311cbe_551147.png)

## 10.01.2019 Версия 2018.3.1 Релиз

Сейчас вышла версия 2018.3.1, поддерживающая последнюю версию IntelliJ IDEA. Основные обновления заключаются в следующем:

- Файл можно отслеживать и видеть только после использования `git add file`.
- Используйте язык Kotlin для написания новых HTTP-интерфейсов.
- Модификация совместимости функциональности IDEA Task.
- Предоставляет функциональные возможности для создания и просмотра запросов на слияние.
